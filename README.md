# TymoMoc

Kolorowa strona edukacyjna dla dzieci w wieku 8 i 9 lat. Zawiera misje z matematyki, języka polskiego, angielskiego i wiedzy o świecie, system XP oraz lokalny zapis postępu.

## Uruchomienie lokalne

Otwórz plik `index.html` w przeglądarce.

## Cloudflare Pages

- Production branch: `main`
- Framework preset: `None`
- Build command: `exit 0`
- Build output directory: `.`

Każdy `git push` do gałęzi `main` uruchamia automatyczne wdrożenie po podłączeniu repozytorium do Cloudflare Pages.
